# GoonerFork

Collection of UserScripts to assist gooners as they goon.

## Included scripts

### Motherless Downloader

This script adds a download button to the Motherless video player, allowing you to easily download videos from the site.

[Add the script to your UserScript manager](https://github.com/coyto/gooner-fork/raw/refs/heads/main/dist/ml-downloader/build.user.js)
